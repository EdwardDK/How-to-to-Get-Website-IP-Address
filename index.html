How to Find Website IP address

  1. First step is to click on the search button, next type in cmd and press enter on the keyboard

  2. For example, type in " tracert www.roblox.com " without the quotes and press enter.

  3. Then, you will see a string of numbers which I have highlighted in red. This is Roblox's IP address

  4. So basically all you need to find an IP of a website is to know the url of it with the www but without https
